Требования
- Небольшое приложение, Qt 5.15, сборка - СМаке.
- Ui: QML (не Widgets).

Что из себя должно представлять:
Приложение, для чтения из csv - файлов пройденных треков и визуализации их на карте.

Фронтенд (Ui):
Одностраничное UI-окно
  - Левая часть - ListModel, с названием трека (можно использовать название csv-файла)
  - Правая часть - Qml-карта с отображением выбранного в модели трека пути.
  - При клике на элемент модели в левой части, в правой части (карте) отображается трек пути и карта центрируется на начале трека. Если до этого был выбран другой трек, этот трек на карте не должен отображаться.

Бекенд:
 - Данные для карты берутся из кастомной директории (можно хардкод), из файлов в формате csv (см. приложения). (Уточню - один файл - один трек).
 - Для чтения файлов можно использовать библиотеки (например, https://github.com/iamantony/qtcsv).
 - Остальные файлы в директории (без расширения .csv) должны игнорироваться.

P.S.: в файлах координаты только во второй и третьей колонке, остальные данные можно игнорировать.
